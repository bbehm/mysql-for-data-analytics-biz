# Intro to databases - with MySQL

### 1️⃣ SQL vs NoSQL

| SQL  | NoSQL |
| ------------- | ------------- |
| For example *MySQL* | For example *mongoDB* |
| Relational  | Non-relational  |
| Use structured query language and have a predefined schema  | Dynamic schemas for unstructured data  |
| Better for multi-row transactions | Better for unstructured data like docs or JSON |

Read more for example [here](https://www.xplenty.com/blog/the-sql-vs-nosql-difference/).

---

### 2️⃣ Structured Query Language

SQL is the language used to create and manipulate relational databases.

SQL is a __declarative__ language, simplified it basically means: *concentrate on what you want done instead of how to do it*.
